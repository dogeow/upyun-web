# upyun-web

技术栈：

- React 19
- Vite 8
- shadcn/ui
- Tailwind CSS 4
- Express 5
- basic-ftp

## 运行

安装依赖后启动开发环境：

```bash
yarn
yarn dev
```

默认地址：

- 前端: `http://127.0.0.1:5173`
- API: `http://127.0.0.1:3001`

生产构建：

```bash
yarn build
```

## 目录

- `src/` React 前端页面和 shadcn 组件
- `server/` 本地 API 和 UpYun/FTP 适配层
- `scripts/dev.js` 前后端联合启动脚本

## 关键交互约定

- 顶部第一行：左侧账号 / 设置抽屉按钮、空间名、状态提示、右侧任务 / 历史抽屉按钮
- 面包屑单独一行
- 根目录在面包屑中显示为 `/`
- 工具栏单独一行
- 固定按钮组：新建文件夹 / 上传、筛选、列表 / 缩略图
- 多选时才显示批量操作按钮组
- 文件列表区域尽量占满可视高度
- 桌面端：单击选中，双击目录进入，双击文件打开详情
- 移动端：单击目录进入，单击文件打开详情
- 工具栏不可用项优先隐藏，不用一排 disabled

## 文件夹下载当前实现

React 版文件夹下载优先使用浏览器目录选择：

- 先弹出目录选择器
- 在所选目录下创建一个同名目录
- 按原目录结构逐个写入文件

如果浏览器不支持，或目录选择失败，则回退为逐个文件下载。

相关代码：

- `src/App.jsx`
- `src/lib/api.js`
- `server/index.js`

## 文件详情与外链

- 文件详情以弹窗展示，不使用侧边栏
- 图片文件会在详情中直接预览
- 文件夹详情展示当前目录的总数、文件夹数、文件数和部分子项样本
- 文件详情中的 `Response Headers` 来自 UpYun `HEAD` 返回，而不是 Express 响应头
- 外链支持按账号保存自定义域名
- 复制链接支持 `URL`、`Markdown`、`HTML` 三种格式

## 任务与历史

- 操作历史只记录目录操作和配置变化，不记录上传 / 下载任务
- 上传任务和下载任务在右侧抽屉中单独展示
- 任务记录保存在本地存储，刷新页面后仍可查看最近任务状态
