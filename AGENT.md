# AGENT.md

## 项目定位

这是 [Updrive](http://github.com/dogeow/updrive) 的 React 重写版

## 技术栈

- React 19
- Vite 8
- shadcn/ui
- Tailwind CSS 4
- Express 5
- basic-ftp

## 目录结构

- `src/App.jsx` 主界面和主要交互逻辑
- `src/lib/api.js` 前端 API 封装
- `src/components/ui/` shadcn/ui 组件
- `server/index.js` 本地 API 服务
- `server/upyun.js` 又拍云 / FTP 适配层
- `scripts/dev.js` 前后端联合启动脚本

## 运行

```bash
npm install
npm run dev
```

默认地址：

- 前端：`http://127.0.0.1:5173`
- API：`http://127.0.0.1:3001`

构建：

```bash
npm run build
```

后端语法检查：

```bash
node --check server/index.js
```

## 当前界面约定

- 顶部第一行：左侧账号 / 设置抽屉按钮、空间名、状态提示、右侧任务 / 历史抽屉按钮
- 面包屑单独一行
- 根目录在面包屑中显示为 `/`
- 工具栏单独一行
- 固定按钮组：新建文件夹 / 上传、筛选、列表 / 缩略图
- 多选时才显示批量操作按钮组
- 文件区域尽量占满可视高度
- 桌面端：单击选中，双击目录进入，双击文件打开详情
- 移动端：单击目录进入，单击文件打开详情
- 不可用工具项优先隐藏，不显示一排 disabled

## 文件夹下载

当前文件夹下载优先走浏览器目录选择：

1. 让用户选择本地目录
2. 在所选目录下创建同名文件夹
3. 按原目录结构逐个写入文件

如果浏览器不支持，或目录选择失败，则回退为逐个文件下载。

相关文件：

- `src/App.jsx`
- `src/lib/api.js`
- `server/index.js`

## 文件详情与外链

- 文件详情用弹窗展示，不走侧边栏
- 图片文件在详情中直接预览
- 文件夹详情展示当前目录的总数、文件夹数、文件数和部分子项样本
- 文件详情中的 `Response Headers` 来自 UpYun `HEAD` 返回
- 外链设置按账号保存在本地
- 复制链接支持 `URL`、`Markdown`、`HTML`

## 任务与历史

- 操作历史记录目录操作和设置变更
- 上传任务和下载任务在右侧抽屉中分开展示
- 上传 / 下载任务记录保存在本地存储，刷新后仍可查看

## 注意事项

- 当前主要维护的是 React 版，不是旧的 Vue / Electron 版
- `node --check` 不能检查 `.jsx`
- 当前目录就是实际维护目录，不需要再同步到别处
